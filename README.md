# YGO Solo-Play Tool

A tool that allows you to play Yu-Gi-Oh OCG as solo. It provides you to following functions.

- Create solo-play gameboard.
- Save gameboard you created as `png`.

## For developers

### Env Requirements

- Node.js 20+

### Run dev server

```bash
$ npm run dev
```

### Build static resources

```bash
$ npm run build
```
